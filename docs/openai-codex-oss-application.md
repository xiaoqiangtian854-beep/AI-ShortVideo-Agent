## Project Description

AI ShortVideo Agent is an early-stage open-source workflow kit for AI-assisted short-video operations. It helps creators, local businesses, and AI builders turn a content brief into a practical production plan, reusable prompts, platform workflows, and review checklists.

The project focuses on short-video planning for platforms such as Douyin, WeChat Channels, TikTok, Bilibili, and enterprise promo videos. It includes a small tested Python planning engine, prompt templates, workflow documentation, and examples that show how AI tools and Codex-assisted development can support content operations.
