# Douyin Operations Workflow

## Goal

Create short videos that quickly communicate value, encourage comments, and move interested viewers into a follow-up channel.

## Workflow

1. Research
   - Pick one audience pain point.
   - Collect three examples of videos in the same niche.
   - Identify the first-frame promise, caption keywords, and comment pattern.

2. Script
   - Write a 3-second hook.
   - Show one concrete problem.
   - Give three practical steps.
   - End with a comment prompt such as "comment checklist".

3. Production
   - Use a clear cover image.
   - Add captions.
   - Keep cuts tight and remove slow context.

4. Publishing
   - Use search-friendly keywords in the caption.
   - Pin a comment that offers the workflow or checklist.
   - Track comments, saves, completion rate, and private traffic leads.

5. Review
   - Keep the hook if retention is strong.
   - Rewrite the first sentence if comments are low.
   - Turn high-comment videos into a content series.
