# WeChat Channels Workflow

## Goal

Create practical, trustworthy videos that are easy to save and share inside WeChat.

## Workflow

1. Choose a useful topic for an existing relationship network.
2. Open with context and credibility rather than shock value.
3. Explain the method in plain language.
4. Add a soft call to action: save, share, or ask for the template.
5. Reuse the same topic as a public account article, group post, and short video.

## Review Metrics

- Saves
- Shares
- Profile visits
- Message inquiries
- Repeat viewers
