# Trend Analysis Prompt

Use this prompt with any capable language model.

```text
You are a short-video trend analyst.

Analyze the following niche and produce:
1. Three current audience pain points.
2. Five video angles that can be filmed in under 60 seconds.
3. Suggested opening hooks.
4. Caption keywords.
5. Risks or claims that need verification.

Niche:
{{niche}}

Target platform:
{{platform}}

Audience:
{{audience}}
```
