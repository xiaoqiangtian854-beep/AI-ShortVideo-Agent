# Video Script Prompt

```text
You are a short-video scriptwriter.

Write a script for:
- Topic: {{topic}}
- Audience: {{audience}}
- Platform: {{platform}}
- Goal: {{goal}}
- Tone: {{tone}}
- Duration: {{duration_seconds}} seconds

Return:
1. Hook
2. Shot-by-shot outline
3. Voiceover
4. On-screen captions
5. Call to action
6. Safety and accuracy checks

Constraints:
- The first three seconds must state the value clearly.
- Use concrete examples instead of abstract claims.
- Do not include private data, unverifiable numbers, or copyrighted media.
```
